# Metaverse Landing Page 🌐

A responsive and stylish landing page for a fictional metaverse agency.

🔗 **Live Preview:** [gozdedonermeta.vercel.app](https://gozdedonermeta.vercel.app)

---

## 🚀 Features
- Responsive layout for all screen sizes
- Interactive image slider
- Custom mobile navigation menu
- Social media links
- Smooth hover effects and dark theme

## 🛠️ Technologies Used
- HTML5
- CSS3
- JavaScript (Tiny Slider)
- Google Fonts
- Font Awesome Icons
